# Contributing to PesDuke

Thanks for your interest in contributing!

## How to Contribute

1. **Fork** the repository
2. **Create** a branch (`git checkout -b feature/my-feature`)
3. **Commit** your changes (`git commit -m "Add my feature"`)
4. **Push** to the branch (`git push origin feature/my-feature`)
5. **Open** a Pull Request

## Development Setup

1. Install [.NET SDK 8.0](https://dotnet.microsoft.com/download/dotnet/8.0)
2. Clone the repo: `git clone https://github.com/fanteamua/PesDuke.git`
3. Open in Visual Studio 2022 or VS Code
4. Build: `dotnet build`
5. Run: `dotnet run` or press F5

## Guidelines

- Keep it simple — this is a small, focused tool
- Dark theme only (yellow-blue accents)
- No external TTS dependencies — use Windows built-in voices
- Test on Windows 10/11 before submitting

## Reporting Issues

- Describe the bug clearly
- Include Windows version and .NET version
- Steps to reproduce
- Expected vs actual behavior
